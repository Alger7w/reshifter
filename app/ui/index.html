ReShifter is a tool and a library for Kubernetes cluster backups. You can use it as a Web app (what you're looking at), you can use it as a CLI tool (rcli) that you can download from the releases page, or you can use it in your own Go codebase as a library.

Start now with Configuration where you can set the etcd endpoint of your Kubernetes cluster, define a remote storage location, as well as explore what's in the Kubernetes cluster. Then you can back up and restore your Kubernetes cluster, either via downloading a ZIP archive or by storing it in the remote storage location. ReShifter currently supports S3-compatible remote storage endpoints, incl. Amazon S3, Google Storage, and Minio.

You can use ReShifter for many different things including but not limited to:

  • Restore When running a Kubernetes cluster in production, you can use ReShifter to back up the current state of the deployed apps and global resources such as namespaces. The backup can be useful as a basis for a restore process (into the same or a new cluster) or for auditing purposes.
  • Disaster Recovery When operating two or more clusters, you can use ReShifter to replicate the state for DR failover scenarios.
  • Saving money When running development or test clusters in a public cloud setting, you might not want to pay for the whole time, only when you actively use it. You can use ReShifter to snapshot the state, shut down the cluster and once ramped up again, restore the state.
  • Troubleshooting For any kind of Kubernetes clusters, be it prod or testing, you can use ReShifter to capture the state of the cluster at a certain point in time for offline debugging and troubleshooting or to share the state with a provider, who then can take care of looking into a support case.
  • Upgrading When you are upgrading a cluster, say, from Kubernetes 1.5 to 1.6, you can use ReShifter to achieve zero-downtime upgrades with minimal manual effort.

Learn more about the origins and philosophy of ReShifter, check out the roadmap or let us know about issues.
