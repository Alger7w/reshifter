Configuration

All settings are stored locally in your browser. This means that while they survive browser re-starts, they are not available across different machines, only to you in this browser.

Default etcd endpoint

Use this as the default for backups and restores.

etcd API version

Use this to define which etcd API version to use for backup, restore, and statistics.

Target environment

Use this setting for backup and restore targets. You've got two options: 1. work locally—that's the local storage of the container ReShifter is running in and if you want to preserve the data beyond the runtime of the container you'll have to mount that volume to some persistent storage such as NFS or EBS— by downloading and uploading the backups as ZIP file, or 2. store the backups in an S3-compatible remote storage.
